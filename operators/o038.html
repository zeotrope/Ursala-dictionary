\   Fix Right Argument
Arity:   0 2 0

  The combinator takes a function as its left operand and a constant as its right operand. The function is expected to be one whose argument is usually a pair of values. The combinator constructs a function that takes only a single value as an argument, and returns the result obtained by applying the original function to the pair made from that value along with the constant operand. 
    f\k x = f(x,k)
