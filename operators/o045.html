Operator:   Functions
--   Composition After Coupling
Arity:   1 3 2

  Takes a function as its left operand and a pair of functions as its right operand, with the algebraic property: 
    f ^(g, h) ≡ f+ (^)(g, h)
   Binary operator lifting: 
    ((f^) (g,h)) x = f(g x,h x)

Solo: 
^s(f,g) = ~&s+ ^(f,g)
Infix: 
f^s(f,g) = f+ ~&s ^(f,g)
Postfix: 
(h^s)(f,g) = h^s(f,g)
   Coupling combinator, takes a pair of functions as an argument and returns a function as a result.This operation corresponds to the virtual machine’s couple combinator. 
     ((^) (f,g)) x = (f x,g x)
